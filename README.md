# -WhatTheMoth-
### Inspired by the challenge featured in Soapy's video.

¡¿WhatTheMoth?! recreates the idea in Hollow Knight with its own implementation.

This is currently a beta version. The mod is still under active development, and I'm working on improving the UI, adding sound effects, translations, and more commands.

How does it work?

Every 15–60 seconds, Radiance will display a random command at the bottom-right of your screen.

It may ask you to:

Dash
Jump
Attack
Cast a spell
Use the Dream Nail
Move in a specific direction
...and more.

The trick?

Don't do what Radiance says.

If you perform the requested action, you'll immediately be sent to Dream_Final (the Radiance arena).
